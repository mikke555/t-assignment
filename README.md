##  Installation
Windows
```
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```
Mac
```
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

##  Usage

```env
Usage: main.py [OPTIONS]

Options:
  -i PATH     Input CSV file contains urls
  -h, --help  Show this message and exit.
```
